# koryphaios Common

Core shared library providing interfaces, base classes, and common utilities for the koryphaios workflow collaboration architecture.

## Overview

This module contains the foundational abstractions that enable the koryphaios event-driven workflow system:
- **Event System**: Tools contribute event handlers that respond to workflow events
- **Tool Adapters**: Plugin architecture for registering custom tools
- **Workflow Models**: Data structures for collaboration workflows
- **Variable Management**: Hierarchical scoping for workflow context
- **Data Resolution**: Interface for querying tool-specific data
